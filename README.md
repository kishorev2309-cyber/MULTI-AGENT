#  Explainable Multi-Agent AI System

## Overview
This project is an **Explainable Multi-Agent AI System** developed for a hackathon.

Unlike traditional AI systems that only give answers, this system shows:
- How decisions are made
- How different agents interact
- Why a particular output is generated

---

## Problem Statement
**Input:** Multi-agent decisions and interactions  
**Output:** Explainability visualization framework  

---

## Architecture

The system uses a **3-agent pipeline**:

### 1️ Planner Agent
- Analyzes user query  
- Identifies intent  
- Extracts keywords  

### 2️ Researcher Agent
- Matches query to a topic  
- Retrieves relevant facts  
- Provides reasoning for topic selection  

### 3️ Decision Agent
- Analyzes facts  
- Generates final answer  
- Assigns confidence score  
- Provides reasoning steps  

---

## Workflow

User Query  
→ Planner Agent  
→ Researcher Agent  
→ Decision Agent  
→ Explainable Output  

---

## Features

- Multi-agent architecture  
- Explainable decision-making  
- Structured reasoning pipeline  
- Confidence scoring  
- Web-based interface  
- Visualization of agent interactions
